Student Performance Analysis

Project Overview
This project performs Exploratory Data Analysis (EDA) on student performance data to understand how different factors such as gender, degree level, and test preparation course impact student scores in math, reading, and writing.

  Objective
- Analyze distribution of student scores
- Compare performance based on degree level
- Evaluate impact of test preparation course
- Identify correlation between subjects
- Detect outliers in score distribution

  Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

  Dataset Features
- Gender
- Degree
- Test Course
- Math Score
- Reading Score
- Writing Score
- Groups

  Analysis Performed
✔ Data Cleaning & Inspection  
✔ Gender Distribution Visualization  
✔ Average Score Analysis by Degree  
✔ Test Preparation Course Analysis  
✔ Correlation Heatmap  
✔ Boxplot for Outlier Detection  
✔ Group Distribution using Pie Chart  

  Key Insights
- Students who completed the test preparation course scored higher on average.
- Higher degree levels were associated with better performance.
- Strong positive correlation observed between reading and writing scores.
- Minor outliers detected in math scores.
